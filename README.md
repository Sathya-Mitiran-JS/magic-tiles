Magic Tiles on Urbana board FPGA

ECE 385 Final Project

Dingjia Shen, Sathya Mitiran



Instructions to run:
1) Load the project in Vivado (final\_project\_refactored.zip), add all design and contraints files

2\) Create a new application project in Vitis and import files from final\_project\_system.zip

3\) Attach hardware system (HDMI, speaker, keyboard) and turn on the FPGA

4\) Build the Vitis project and run
